This is an App developed for Devscale Tutorial.
If you wanted to use this repo for learning please do this.

1. Install the Dependencies

```bash
npm install
```

2. Generate the Prisma Client

```bash
npx prisma generate
```

3. Run the app

```bash
npm run dev
```

The DEMO is here :
https://space-travel-devscale.vercel.app/
